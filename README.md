## 5.3 （雨林 + 泼水）：中科院热带植物园 → 傣族园泼水

### 上午：中科院热带植物园（8:00–12:00）

- [ ] 门票：80 元，西区景区车：50元，东区景区车：50元
  - 路线：先东区（原始雨林）→ 再西区（王莲、奇花异草）
  - 看点：大板根、绞杀树、王莲池（可坐人）

### 下午：傣族园（13:30–17:00）

- [ ] 门票：45 元，14:50/16:30 两场泼水
  - 可租傣装（30 元）水盆，加入百人泼水大战
  - 路逛古村寨、佛寺，体验傣家生活

### 晚上：回告庄，吃傣味手抓饭 / 烧烤，早点休息

- 提示：这天是东线一日，路程稍远，建议早点出门；做好防晒 + 防蚊。

## 5.4 （市区人文 + 大象）：曼听公园 + 总佛寺 → 野象谷

### 上午：曼听公园 & 总佛寺（8:30–12:00）

  - [ ] 门票 40 元，傣王御花园，建筑很出片
